### S-01 스플래시 화면

**목적:** 앱 초기 로딩 중 표시  
**표시 시간:** 최대 2초

**레이아웃:**
- 전체 배경: 포인트 컬러(#2563EB) 단색
- 중앙: 앱 로고 (흰색, 48x48px 아이콘 + "송줍줍" 워드마크)
- 하단 중앙: 로딩 인디케이터 (흰색 점 3개 애니메이션)

**로직:**
- 자동 로그인 토큰 확인 → 토큰 있으면 S-04 홈으로 이동
- 토큰 없으면 S-02 온보딩 (첫 실행) 또는 S-03 로그인으로 이동

### S-02 온보딩 화면 (3페이지)

**목적:** 앱 첫 실행 시 핵심 가치 전달  
**형태:** 스와이프 가능한 페이지 슬라이더

**페이지 1 — AI & 채용 뉴스 자동 수집**
- 상단 일러스트 영역 (280x240px): 뉴스 카드들이 쌓이는 애니메이션 이미지
- 제목 (24px, Bold): "최신 AI · 채용 뉴스를\n자동으로 모아드려요"
- 설명 (15px, Regular, Secondary Color): "매일 아침 업데이트되는\nAI 뉴스와 채용 정보를 한눈에"

**페이지 2 — 나만의 스크랩 저장**
- 상단 일러스트 영역: 북마크하는 동작 이미지
- 제목: "읽고 싶은 콘텐츠를\n내 창고에 저장하세요"
- 설명: "URL 공유만 해도 자동으로\n제목·요약·이미지를 저장해드려요"

**페이지 3 — 채용 마감 알림**
- 상단 일러스트 영역: 알림 뱃지 이미지
- 제목: "채용 공고 마감일을\n절대 놓치지 마세요"
- 설명: "D-7, D-3, D-1 푸시 알림으로\n중요한 공고를 챙겨드려요"

**공통 하단 요소:**
- 페이지 인디케이터 (점 3개, 활성 점: 포인트 컬러)
- "시작하기" 버튼 (Full Width, 포인트 컬러, 마지막 페이지에만 표시)
- "건너뛰기" 텍스트 버튼 (우상단, Secondary Color)

### S-03 로그인 화면

### S-04 홈 피드 화면

**목적:** 전체 카테고리 최신 스크랩 통합 피드  
**탭바 위치:** 하단 (홈 탭 활성)

**헤더 영역 (56px):**
- 좌측: "송줍줍" 로고 텍스트 (18px, Bold)
- 우측: 검색 아이콘(24px) + URL 추가 아이콘(24px)

**카테고리 필터 탭 (수평 스크롤 가능, 높이 44px):**
- 탭 목록: 전체 · AI 뉴스 · 채용 뉴스 · 채용 공고 · 재테크
- 활성 탭: 포인트 컬러 텍스트 + 하단 2px 바 강조
- 비활성 탭: Secondary Color 텍스트
- 좌우 패딩: 16px, 탭 간 간격: 24px

**피드 영역 (무한 스크롤):**
- 좌우 패딩: 16px
- 카드 간 간격: 12px

**뉴스 카드 컴포넌트 (기본형):**
```
┌─────────────────────────────────────┐
│ [카테고리 뱃지]      [출처] · [N분 전] │  ← 상단 메타 (13px)
│                                     │
│ 기사 제목 (16px, SemiBold, 최대 2줄) │
│                                     │
│ 요약 텍스트 (14px, Regular,          │
│ Secondary, 최대 2줄)                 │
│                          [썸네일]   │
│                          (80x80px)  │
│                          (우측 정렬) │
├─────────────────────────────────────┤
│ [읽음여부 점] [태그1][태그2]  [북마크]│  ← 하단 액션 (12px)
└─────────────────────────────────────┘
```

**뉴스 카드 상세 스펙:**
- 카드 배경: 흰색(라이트) / #2C2C2E(다크)
- 카드 패딩: 16px
- 카테고리 뱃지: 해당 카테고리 컬러 배경, 흰색 텍스트, 12px, 4px 4px 패딩
- 제목: 16px, SemiBold, Primary Color, 2줄 말줄임
- 요약: 14px, Regular, Secondary Color, 2줄 말줄임
- 썸네일: 80x80px, border-radius 8px, 우측 상단 배치, 이미지 없으면 카테고리 컬러 배경 + 아이콘
- 출처: 13px, Tertiary Color
- 시간: 13px, Tertiary Color (N분 전 / N시간 전 / 날짜)
- 읽음 표시: 좌측 하단 6px 원 (읽지 않음: 포인트 컬러, 읽음: 투명)
- 태그: 12px, pill 형태, 배경 Surface Color
- 북마크 아이콘: 24px, 우측 하단 (활성: 포인트 컬러, 비활성: Tertiary)

**채용 공고 카드 컴포넌트 (D-Day 강조형):**
```
┌─────────────────────────────────────┐
│ [회사 로고 32px] 회사명   [D-3 뱃지] │
│               포지션명               │
│ 경력: N년 이상  │  서울              │
│ [기술스택 태그들]                     │
│─────────────────────────────────────│
│ [북마크] [원문 보기 →]               │
└─────────────────────────────────────┘
```

**D-Day 뱃지 컬러:**
- D-7 이상: Secondary Color (회색)
- D-3 ~ D-6: 경고색 (#F59E0B)
- D-1 ~ D-2: 위험색 (#EF4444)
- 마감: Tertiary Color + 취소선

**피드 상단 고정 요소:**
- 오늘의 업데이트 요약 바 (44px): "오늘 N개의 새 콘텐츠가 업데이트됐어요" + 새로고침 아이콘

**하단 탭바:**
- 탭: 홈(house 아이콘) · AI(robot 아이콘) · 채용(briefcase 아이콘) · 재테크(chart 아이콘) · MY(person 아이콘)
- 활성 탭: 아이콘 + 텍스트 라벨(10px), 포인트 컬러
- 비활성 탭: 아이콘만, Tertiary Color

## 인터랙션 & 애니메이션 가이드

### 화면 전환
- 하단 탭 전환: 슬라이드 없이 즉시 전환 (Fade)
- 카드 탭 → 상세: 오른쪽에서 슬라이드 인 (표준 iOS/Android 패턴)
- Bottom Sheet: 아래에서 슬라이드 업 (duration: 300ms, ease-out)
- 모달: 중앙에서 Scale + Fade In (duration: 250ms)

### 피드백 애니메이션
- 카드 탭: scale(0.97) 순간 눌림 효과 (duration: 100ms)
- 북마크 토글: 하트 아이콘 scale(1.3) → scale(1.0) 바운스 (duration: 200ms)
- 스크랩 저장 완료: 하단 토스트 메시지 (2초 후 사라짐)
  - "저장됐어요 ✓" (체크 아이콘 + 텍스트, 포인트 컬러 배경, 흰색 텍스트)

### 로딩 상태
- 피드 초기 로딩: 스켈레톤 카드 3개 (카드 형태 + 회색 박스 + shimmer 애니메이션)
- 페이지네이션 로딩: 하단 중앙 스피너 (포인트 컬러)
- OG 메타데이터 로딩: 스켈레톤 프리뷰

### 당겨서 새로고침 (Pull to Refresh)
- 피드 화면 전체 적용
- 인디케이터: 포인트 컬러 원형 스피너
